# Economy

Economy 是一款虚拟市场经营策略游戏。

当前优先开发：**抖音小程序单机版**。

第一阶段不做多人联机、不做服务器、不做 Steam 库存、不做共享市场，而是先在抖音小程序中做出完整单机可玩循环。

## 核心循环

```text
点击工作 -> 获得劳动力 -> 兑换货物 -> 出售货物 -> 获得金融货币 -> 升级设施 -> 提升效率
```

## 最小可玩版本

第一版只做最小经济链路：

1. 点击工作获得劳动力。
2. 使用 3 劳动力生产 1 食品包。
3. 出售 1 食品包获得 5 金融货币。
4. 使用金融货币升级工作席位。
5. 工作席位提升点击获得的劳动力。
6. 使用本地存档保存进度。

## 当前开发决策

- 游戏名称：Economy
- 优先平台：抖音小程序
- 优先版本：单机版
- 第一阶段不需要服务器
- 第一阶段不需要 Steam 库存
- 第一阶段不需要多人联机
- 第一阶段不需要玩家列表
- 第一阶段不需要创建房间
- 点击不直接产出金融货币
- 点击产出劳动力
- 劳动力用于生产货物
- 货物出售获得金融货币
- 金融货币用于升级设施

## 后续扩展

单机版跑通后再考虑：

- 更多货物类型
- 本地价格波动
- 成就系统
- 每日任务
- 抖音平台能力接入
- 后端服务器版本
- 多人共享市场
- Steam 桌面版本

## 原桌面技术栈

此前项目参考 `RIVERS0FT/TheGenius` 的技术路线：

- Tauri 2.x
- Rust
- React
- TypeScript
- Vite
- Zustand

这些内容暂时后置。后续可将核心玩法逻辑抽成独立模块，再复用到桌面端或服务器版本。
