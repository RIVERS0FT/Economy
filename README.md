# Economy

Economy 是一款虚拟市场经营策略游戏。

当前优先开发：**抖音小程序单机版**。

小程序版显示名称：**我不是股神**。

第一阶段不做多人联机、不做服务器、不做 Steam 库存、不做共享市场，而是先在抖音小程序中做出完整单机可玩循环。

小程序版只聚焦手机移动端体验，不适配桌面端、大屏端或复杂横向布局。

## 当前实现

抖音小程序单机最小可玩版本已实现，代码位于：

```text
miniprogram/
```

已完成链路：

```text
点击工作 -> 获得劳动力 -> 生产食品包 -> 出售食品包 -> 获得金融货币 -> 升级工作席位
```

## 已实现功能

- 点击工作获得劳动力
- 点击冷却：0.2 秒
- 点击不直接获得金融货币
- 3 劳动力生产 1 食品包
- 1 食品包出售获得 5 金融货币
- 金融货币升级工作席位
- 工作席位升级后提升点击收益
- 年龄信息
- 座驾信息与升级
- 房子信息与升级
- 本地存档
- 操作日志
- 重置存档
- 移动端单列优先布局

## 运行方式

1. 打开抖音开发者工具。
2. 导入 `miniprogram` 目录。
3. 在 `miniprogram/project.config.json` 中填写自己的小程序 `appid`。
4. 编译运行。

## 当前开发决策

- 游戏名称：Economy
- 小程序版名称：我不是股神
- 优先平台：抖音小程序
- 优先版本：单机版
- 小程序版只聚焦移动端
- 第一阶段不需要服务器
- 第一阶段不需要 Steam 库存
- 第一阶段不需要多人联机
- 第一阶段不需要玩家列表
- 第一阶段不需要创建房间
- 点击不直接产出金融货币
- 点击产出劳动力
- 劳动力用于生产货物
- 货物出售获得金融货币
- 金融货币用于升级设施、座驾和房子
