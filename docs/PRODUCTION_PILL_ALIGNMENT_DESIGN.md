# Economy 生产页胶囊与开关对齐设计

> 状态：生产页一级卡片右上角胶囊几何基线  
> 适用项目：`RIVERS0FT/Economy`  
> 更新时间：2026-07-15

本设计补充 `UI_DESIGN_SYSTEM.md`、`WAREHOUSE_EXPANSION_DESIGN.md` 与 `INDUSTRY_AND_PRODUCTION_DESIGN.md`。

## 1. 统一对象

生产页统一以下可见胶囊几何：

- `StatusTag` 状态胶囊；
- 共享仓库等级胶囊；
- 工厂集群 `SwitchControl` 的可见轨道。

状态与等级继续使用 `StatusTag`，开关继续使用唯一的 `SwitchControl`。

## 2. 可见几何

生产页工厂开关可见轨道与状态、等级胶囊共享：

```text
可见高度：1.6rem
圆角：var(--radius-pill)
边框：1px
```

状态与等级胶囊宽度由文字决定，开关轨道保持固定宽度。统一的是高度、圆角、边框厚度和卡片边缘基准。

## 3. 点击区域

`SwitchControl` 的点击区域继续保持至少 `44 × 44px`。可见轨道从点击区域顶部开始，点击区域剩余空间只向下延伸。因此工厂开关轨道与仓库等级胶囊位于卡片右上角时，可见顶部和右边缘使用相同的卡片内边距。

点击区域不得缩小到可见轨道高度。标题行仍以 44px 点击区域参与布局，标题保持顶部对齐。

## 4. 滑块与焦点

滑块保持 `1rem`，并使用以下关系垂直居中：

```css
top: calc((可见轨道高度 - 滑块尺寸) / 2);
```

开启状态左右留白必须一致。焦点环继续绘制在 `.ui-switch::before` 可见轨道外侧，不得包围整个 44px 点击区域。

## 5. 样式职责

- `design-system.css` 定义 `StatusTag`、全局开关基础外观和焦点环；
- `production-surface.css` 负责生产页同平面卡片中的开关轨道对齐；
- `facility-group-card-grid.css` 负责工厂标题、状态和数量摘要布局，不得重新设置开关伪元素垂直位置。

## 6. 防回退

不得：

- 把生产页工厂开关轨道恢复为在 44px 点击区域内垂直居中；
- 让开关轨道高度不同于 `StatusTag` 的 `1.6rem`；
- 缩小 44 × 44px 点击区域；
- 用额外外边距单独修补某一种工厂卡；
- 让开关点击区域下压标题；
- 把焦点环恢复为包围整个点击区域的大圆环；
- 未同步更新本文档和验证脚本就改变生产页胶囊几何。
