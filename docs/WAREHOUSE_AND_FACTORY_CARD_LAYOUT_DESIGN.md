# Economy 共享仓库与工厂卡片布局设计

> 状态：生产页面正式布局与防回退基线

## V3：左侧常驻建设卡、固定工厂卡与无限仓库（2026-07-12）

- 建设新工厂卡独占生产管理区左侧列，桌面使用 `position: sticky`，移动端恢复 `position: static`。
- 工厂卡桌面固定高度 384px，移动端自然高度；开关始终对齐右上角。
- 运行状态不显示“正常生产中”说明；剩余周期只在进度区出现。
- 异常原因直接合并为“异常：仓库已满”“异常：资金不足”“异常：原料不足”等顶部状态。
- 标题显示总数量；正文只在顶部摘要显示“运行中／下一周期加入／冻结中”，不得恢复独立数量卡片。
- 删除“下一周期按若干座结算”说明。
- 计划区只保留当前计划、模式下拉框、定量输入和“保存计划”按钮，不得恢复持续／定量状态小卡片。
- 定量生产完成后关闭开关；若存在待生效计划，先保存为新当前计划，但仍保持停止，等待玩家再次开启。
- 仓库无限扩容，每级容量增量随等级增加，等级标题不得出现最高等级。
