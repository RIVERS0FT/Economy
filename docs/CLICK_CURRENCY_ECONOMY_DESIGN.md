# Economy 点击货币与冷却经济设计

> 状态：核心货币发行规则基线  
> 当前客户端：网页端  
> 服务器约束：2 核 2G  
> 更新时间：2026-07-10

## 1. 规则概述

玩家点击“工作”后直接获得货币，不设置劳动力、体力或能量资源。

```text
点击工作
-> 服务器确认冷却结束
-> 玩家获得货币
-> 建造或购买工厂
-> 工厂生产基础商品
-> 在统一市场交易商品与工厂
-> 扩大企业资产
```

点击收入用于提供启动资金和基础兜底收入，工厂经营与市场交易应成为中后期主要收益来源。

## 2. 点击规则

第一轮测试建议：

| 项目 | 建议值 |
|---|---:|
| 每次有效点击收入 | 1 货币 |
| 基础冷却 | 3 秒 |
| 连续工作冷却档位 | 3 / 5 / 8 / 12 秒 |
| 无操作恢复时间 | 5 分钟 |

建议连续工作规则：

- 第 1～20 次有效点击：3 秒。
- 第 21～40 次：5 秒。
- 第 41～60 次：8 秒。
- 第 61 次及以后：12 秒。
- 连续 5 分钟没有有效点击后恢复基础冷却。

所有数值都需要根据试玩和经济数据调整。

## 3. 点击状态

- 可以工作
- 冷却中
- 正在确认
- 网络中断
- 工作暂时暂停

冷却期间再次点击：

- 不增加货币。
- 不重复提交正式收入。
- 不能伪装成成功。

## 4. 服务器确认规则

服务器负责：

- 判断当前点击是否满足冷却。
- 维护连续工作冷却档位。
- 确认本次点击收入。
- 增加正式可用货币。
- 写入资金流水。
- 防止刷新、多标签页和重复请求绕过限制。

客户端只负责倒计时、按钮状态和动画展示。

## 5. 货币来源

MVP 增加全服货币的来源：

1. 点击工作。
2. 虚拟人口购买基础商品。
3. 有限的新手任务或运营奖励。

玩家之间的商品交易和工厂交易只转移已有货币，不增加全服货币总量。

## 6. 货币回收

MVP 货币回收包括：

- 工厂建造费用
- 工厂运营费用
- 工厂升级费用
- 企业扩建和工厂槽位费用
- 企业仓库升级费用

后续可以增加适度市场服务费、许可证和维护费用，但不能让新玩家无法启动。

## 7. 点击收入与工厂成长

点击收入主要承担三个作用：

- 帮助新玩家积累第一座工厂的资金。
- 在经营失败或暂时没有工厂时提供恢复途径。
- 为短时上线玩家提供确定性进度。

玩家拥有工厂后，单位时间内的经营利润应逐渐高于点击收入，使玩家自然转向：

- 建造工厂
- 收购工厂
- 生产基础商品
- 商品买卖
- 工厂资产交易

## 8. 点击收入与工厂价格

第一座自建工厂的费用和点击收入必须共同测试。

目标：

- 新玩家可以在合理时间内接近第一座工厂。
- 不能通过持续点击在短时间内大量收购工厂。
- 工厂经营失败后仍能通过点击重新启动。
- 购买现成工厂的便利应由更高市场价格体现，而不是额外等待。

当前建议基础工厂建造费用为 60 货币、施工时间为 5 分钟，仅作为首轮测试基线。

## 9. 通胀控制

点击会直接发行货币，因此需要同时监测：

- 每名玩家每小时有效点击数
- 每名玩家每小时点击收入
- 全服点击发行货币量
- 人口消费发行货币量
- 工厂建造回收量
- 工厂运营回收量
- 工厂升级和扩建回收量
- 全服货币净增长量
- 基础商品成交价格
- 工厂成交价格

货币增长过快时，调整顺序建议：

1. 延长连续工作冷却。
2. 降低点击收入。
3. 调整人口消费预算。
4. 调整建造、运营和升级回收。

不得通过删除玩家资产或修改已完成交易处理通胀。

## 10. 防作弊

- 冷却以服务器时间为准。
- 修改客户端时间不能提前点击。
- 同一点击请求最多成功一次。
- 多标签页共享同一冷却。
- 刷新不能重置冷却。
- 断线期间不补发点击收入。
- 不提供离线自动点击。
- 点击异常时可以暂停工作功能，但不能影响已有工厂、订单和资产。

## 11. 2 核 2G 约束

- 点击不是全服广播事件。
- 其他玩家不会看到单次点击。
- 点击只更新当前玩家余额和流水。
- 冷却期间重复操作不应形成大量有效请求。
- 点击流水分页加载。
- 服务器压力较高时可以临时延长全服基础冷却，并在页面明确提示。

## 12. 页面交互

工作区域展示：

- 可用与冻结资金
- 本次点击收入
- 工作按钮
- 冷却进度与剩余时间
- 当前连续工作冷却档位
- 恢复基础冷却的剩余时间
- 本周期点击收入统计
- 建造工厂和统一市场快捷入口

成功提示：

```text
工作完成，获得 1 货币
```

冷却提示：

```text
工作冷却中，2.1 秒后可再次工作
```

## 13. 新手引导

1. 点击工作获得第一笔货币。
2. 查看点击收入流水。
3. 观察点击冷却。
4. 积累工厂资金。
5. 选择自建工厂或查看玩家工厂挂牌。
6. 自建则等待施工，购买则立即获得工厂。
7. 启动工厂生产。
8. 在统一市场出售基础商品。
9. 使用收益升级、扩建或收购工厂。

## 14. 验收标准

1. 游戏中不存在劳动力资源。
2. 每次有效点击直接增加货币。
3. 冷却未结束时点击不增加货币。
4. 修改本地时间、刷新或多标签页不能重置冷却。
5. 每次点击收入都有资金流水。
6. 点击收入能够支持玩家逐步接近第一座工厂。
7. 点击不能直接产生基础商品或工厂。
8. 工厂建造、运营和升级能够回收货币。
9. 玩家间商品和工厂交易不增加全服货币总量。
10. 人口消费发行量受到周期预算限制。
11. 点击功能暂停不会影响工厂、市场和已有资产。
12. 中后期主要收入能够从点击转向经营和交易。

## 15. 当前开发决策

- 点击直接获得货币。
- 点击收益和冷却由服务器确认。
- 连续工作会提高冷却。
- 不设置劳动力、体力或离线点击收入。
- 点击收入用于启动和兜底。
- 工厂建造、运营、升级和扩建是主要货币回收方式。
- 工厂经营和统一市场交易是中长期主要成长方式。
