# Economy 页面内容与导航职责设计

> 状态：当前正式页面内容、操作归属与导航基线  
> 适用项目：`RIVERS0FT/Economy`  
> 更新时间：2026-07-11  
> 客户端状态版本：`version: 8`

工厂集群和市场第三版细则以 `docs/FACILITY_GROUP_AND_MARKET_V3_DESIGN.md` 为准。

## 1. 页面划分原则

```text
概览：了解当前状态并进入主要流程
市场：完成商品和工厂数量交易
生产：管理共享仓库、建设工厂集群和统一控制生产
资金：查看资产结果与本地变化记录
排行：比较服务器总资产
设置：管理资料、客户端偏好、登录会话和危险操作
```

每个正式操作必须有唯一主要页面；完整管理模块不得跨页重复；全局状态栏只提供只读摘要；设置页不是经营仪表盘。

## 2. 正式导航

```text
概览｜市场｜生产｜资金｜排行｜设置
```

| 导航 | 路由 ID | React 页面 | 职责 |
|---|---|---|---|
| 概览 | `home` | `OverviewPage` | 摘要与快捷入口 |
| 市场 | `market` | `MarketPage` | 商品和工厂数量交易 |
| 生产 | `production` | `ProductionPage` | 共享仓库与工厂集群管理 |
| 资金 | `assets` | `AssetsPage` | 资产结果与本地记录 |
| 排行 | `leaderboard` | `LeaderboardPage` | 总资产排名 |
| 设置 | `settings` | `SettingsPage` | 资料、偏好、会话和重置 |

本文中的“工厂页面”指用户从“生产”导航进入的 `ProductionPage`。

不得恢复独立订单页、成交记录页、仓库页、工厂实例页、`records` 路由或设施槽位页面。

## 3. 应用外壳

`GameShell` 统一提供桌面侧栏、移动底部导航、全局状态栏、操作提示和页面滚动区域。

状态栏可以显示资金、总资产、排名、选中商品持仓、共享仓库使用与买单预占、最近成交价。不得在状态栏提供扩容、下单、撤单、建厂、生产计划或设置操作。

## 4. 概览页面

必须包含：

- 玩家问候；
- 基础工作；
- 当前商品市场摘要与六种商品快捷行情；
- 工厂总数、当前参与、下一周期加入、阻塞、施工、定量计划和共享仓库剩余；
- 财富摘要；
- 当前浏览器最近成交；
- 市场、生产和资金快捷入口。

不得包含完整订单簿、完整工厂集群卡、仓库扩容卡、设置表单或完整排行榜。

## 5. 市场页面

必须包含：

- 商品选择与行情；
- 买入／卖出方向；
- **限价输入位于数量输入之前**；
- 数量输入；
- 数量下方的 `1/4 仓`、`1/2 仓`、`全仓` 快捷按钮；
- 当前商品订单与快捷撤单；
- 单列订单簿；
- 最优 5 笔卖单、最近成交与价差、最优 5 笔买单；
- 价格曲线；
- 全部未完成订单；
- 冻结资金、冻结商品和买单预占摘要；
- 本地成交记录；
- 工厂数量挂牌、数量购买和撤销挂牌。

买入快捷数量受资金和共享仓库剩余空间共同限制。卖出快捷数量只按当前商品可用库存计算。

订单簿每行只显示方向、价格和剩余数量，不显示玩家身份，不进行价格档位聚合。

市场不得包含仓库扩容、建设工厂、工厂启停、生产计划、资产配置或账号设置。

## 6. 生产页面（工厂页面）

### 6.1 页面顺序

```text
页面标题与工厂状态
→ 共享仓库
→ 建设新工厂
→ 工厂类型集群卡
```

共享仓库必须位于建设与集群卡之前。`WarehouseUpgradeCard` 只能由 `ProductionPage` 渲染。

### 6.2 共享仓库内容

必须显示等级、已用容量、实物库存、买单预占、剩余容量、扩容后容量、扩容费用、扩容按钮、资金不足、容量超限和满级状态。

### 6.3 建设新工厂内容

必须显示：

- 工厂类型与配方；
- 建造费用和施工时间；
- 生产周期、单座周期产量和单座周期成本；
- 产成品直接进入共享仓库；
- 当前施工任务和剩余时间；
- 新工厂在下一周期加入的说明；
- 建设按钮；
- 同时只能施工一座工厂的说明。

### 6.4 工厂类型集群卡

每种 `facilityTypeId` 最多一张卡，不支持展开实例。

必须显示：

- 工厂类型和持有数量；
- 统一状态与停止原因；
- 当前参与数量；
- 下一周期参与数量；
- 待加入数量；
- 已挂牌数量；
- 统一周期进度；
- **周期**；
- **周期产量**；
- **周期成本**；
- **原料库存**；
- 统一持续或定量计划；
- 启动全部未挂牌工厂或停止当前生产集群；
- 挂牌工厂不参与生产，已有挂牌不阻止未挂牌数量启动；
- 数量挂牌、单座价格和挂牌总额；
- 产成品自动入仓说明。

不得显示：

- 工厂实例名称、ID、列表或展开实例；
- 单座启停；
- 小时产量；
- 小时运营费；
- 累计产量；
- 系统参考估值；
- 内部商品、内部容量或领取按钮。

宽屏集群卡双列，紧凑桌面、平板和移动端单列。

### 6.5 新数量加入

施工完成或市场收购发生在集群运行期间时：

- 当前周期继续使用旧参与数量；
- 新数量显示为待加入；
- 下一周期参与数量包含待加入数量；
- 当前周期完成后加入；
- 不重置进度；
- 不产生部分周期产出。

### 6.6 阻塞说明

`full` 和 `output_full` 表示共享仓库无法容纳下一个集群完整周期的净新增商品。扩容、补充资金或补充原料后不会自动重启。

## 7. 资金页面

必须包含：

- 可用资金、冻结资金和总资产；
- 商品资产与工厂数量资产；
- 资产配置与估值明细；
- 商品库存与估值；
- 货币发行与回收；
- 当前浏览器本地资金、商品、仓库、工厂集群和生产变化；
- 本地事件筛选与清除。

工厂资产按类型数量乘以服务器系统估值计算。资金页不负责仓库管理，不得渲染扩容卡、建厂、生产控制或下单表单。

## 8. 排行页面

只读展示服务器总资产排名，包含我的排名、总资产、与上一名差距、本周变化和完整排行榜。不得包含交易、生产、仓库扩容或设置操作。

## 9. 设置页面

设置页只允许：

- 玩家资料与昵称；
- 客户端界面偏好；
- 登录会话与退出登录；
- 重置服务器经济状态。

不得显示账号与产业摘要、仓库使用摘要、工厂、商品、订单或排名经营状态。

## 10. 模块唯一归属

| 模块 | 唯一主要页面 |
|---|---|
| 基础工作 | 概览 |
| 商品下单、快捷仓位、单列订单簿与撤单 | 市场 |
| 工厂数量交易 | 市场 |
| 共享仓库扩容 | 生产 |
| 建厂、统一计划、统一进度和统一启停 | 生产 |
| 工厂数量挂牌 | 生产 |
| 资产配置与本地变化 | 资金 |
| 总资产排名 | 排行 |
| 玩家资料、偏好、会话和重置 | 设置 |

不再存在工厂实例管理或领取产成品模块。

## 11. 防回退规则

不得：

- 在生产页之外渲染完整仓库管理卡；
- 恢复工厂实例、实例 ID、实例列表或单座操作；
- 恢复内部容量、内部商品或领取按钮；
- 在工厂卡恢复小时产量、小时运营费、累计产量或系统参考估值；
- 把数量输入放到限价之前；
- 把快捷仓位按钮移出数量输入下方；
- 恢复双列订单簿或显示超过 5 笔买／卖盘；
- 在资金页提供扩容或生产操作；
- 在设置页恢复经营摘要；
- 修改页面职责但不更新设计和检查。

未更新本设计、第三版专题设计和防回退检查的页面结构修改不应合并。
