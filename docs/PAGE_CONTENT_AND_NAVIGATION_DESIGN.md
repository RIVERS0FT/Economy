# Economy 页面内容与导航职责设计

> 状态：当前正式页面、模块唯一归属与导航基线  
> 适用项目：`RIVERS0FT/Economy`  
> 更新时间：2026-07-12  
> 客户端状态版本：10  
> 世界状态版本：6

## 1. 正式导航

```text
概览｜市场｜生产｜资金｜排行｜设置
```

| 导航 | 路由 ID | React 页面 | 唯一职责 |
|---|---|---|---|
| 概览 | `home` | `OverviewPage` | 工作和经营摘要 |
| 市场 | `market` | `MarketPage` | 商品与工厂统一交易 |
| 生产 | `production` | `ProductionPage` | 仓库、建设和工厂集群 |
| 资金 | `assets` | `AssetsPage` | 资产结果与本地变化 |
| 排行 | `leaderboard` | `LeaderboardPage` | 服务器总资产排名 |
| 设置 | `settings` | `SettingsPage` | 资料、偏好、礼品、退出和重置 |

不得恢复独立订单页、成交记录页、仓库页、工厂实例页、`records` 路由或设施槽位页面。

## 2. 应用外壳

`GameShell` 提供：

- 桌面侧栏；
- 移动底部导航；
- 全局状态栏；
- 操作结果提示；
- 唯一页面纵向滚动区域。

状态栏固定显示可用资金、总资产、排行榜和仓库剩余。状态栏只读，不提供扩容、下单、撤单、建厂、计划或设置操作。

## 3. 概览

必须包含：

- 玩家问候；
- 基础工作和 10 秒冷却；
- 按服务器目录生成的商品行情摘要；
- 工厂运行、停止、异常、施工和待加入摘要；
- 财富摘要；
- 当前浏览器最近成交；
- 前往市场、生产和资金的入口。

不得包含完整订单簿、完整工厂卡、仓库扩容、设置表单或完整排行榜。

## 4. 市场

必须包含：

- 商品和工厂统一资产标签；
- 买入／卖出方向；
- 限价；
- 数量；
- 数量下方快捷仓位；
- 订单总额和提交；
- 当前资产未完成订单与撤单；
- 全部自有未完成订单；
- 单列最优 5 笔卖单、中性分隔线和最优 5 笔买单；
- 价格曲线；
- 当前浏览器本地成交记录。

不得包含：

- 商品／工厂二级页面切换；
- 工厂固定价格挂牌或购买卡；
- 玩家身份出现在订单簿；
- 价格档位聚合；
- 仓库扩容、建设、启停或计划编辑；
- 内部固定高度、分页或垂直滚动的本地成交面板。

## 5. 生产

页面顺序：

```text
标题与工厂状态汇总
→ 共享仓库
→ 建设新工厂
→ 工厂类型集群卡
```

### 5.1 共享仓库

显示等级、容量、实物库存、买单预占、剩余容量、扩容信息和动态商品卡。完整规则见 `WAREHOUSE_EXPANSION_DESIGN.md`。

### 5.2 建设

显示：

- 工厂类型和配方；
- 建造费和施工时间；
- 周期、单座产量和成本；
- 当前施工任务；
- 产成品直接入仓；
- 新数量下一周期加入；
- 同时只能施工一座。

### 5.3 工厂集群卡

每种工厂类型最多一张卡。必须显示：

- 工厂类型和持有数量；
- 运行、停止或异常状态及原因；
- 参与、待加入、下一周期和冻结数量；
- 周期进度、产量、成本和原料；
- 当前计划、完成量和待生效计划；
- 统一运行开关；
- 前往统一订单簿的工厂市场入口。

不得显示工厂实例、实例 ID、内部库存、领取、单座启停、固定价格挂牌、小时指标或系统参考估值。

## 6. 资金

必须包含：

- 可用资金、冻结资金和总资产；
- 商品资产；
- 工厂数量资产；
- 服务器估值价格与资产组成；
- 货币发行和系统回收；
- 当前浏览器本地资金、商品、仓库、工厂和生产变化；
- 本地事件筛选与清除。

资金页不负责仓库管理、建厂、生产控制或下单。

## 7. 排行

只读展示：

- 我的排名；
- 总资产；
- 与上一名差距；
- 本周变化；
- 完整服务器排行榜。

不得包含交易、生产、扩容或设置操作。

## 8. 设置

设置页只允许：

- 玩家资料与四项统计；
- 客户端偏好；
- 礼品兑换；
- 管理员入口，仅 `role=admin` 显示；
- 退出登录；
- 重置经济状态。

不得作为经营仪表盘，不显示工厂、商品、订单、仓库或排名摘要。

## 9. 商品与工厂目录扩展规则

所有目录型页面必须遍历服务器返回的 `products` 和 `facilityTypes`。

- 不得写死 6 个商品 ID。
- 不得固定六个行情卡、六个商品标签或六张工厂卡。
- 新商品和工厂应在不修改页面结构的情况下自动出现。
- 目录型横向导航可以滚动或自动分配列宽，但不能依赖固定项目数量。
- 空目录必须显示明确空状态。

## 10. 模块唯一归属

| 模块 | 唯一主要页面 |
|---|---|
| 工作 | 概览 |
| 统一资产下单、订单簿、撤单和本地成交 | 市场 |
| 仓库管理和扩容 | 生产 |
| 建设、周期、计划、启停和工厂状态 | 生产 |
| 资产组成与本地资产事件 | 资金 |
| 总资产排名 | 排行 |
| 资料、偏好、礼品、退出和重置 | 设置 |
| 礼品码管理 | 管理员页面 |

## 11. 防回退

不得：

- 跨页复制完整管理模块；
- 恢复独立订单、记录、仓库或工厂实例路由；
- 恢复固定价格工厂市场；
- 恢复六商品写死页面；
- 把仓库扩容移出生产页；
- 把计划或启停移到市场页；
- 给本地成交表添加内部纵向滚动、分页或截断；
- 在设置页恢复经营摘要。
